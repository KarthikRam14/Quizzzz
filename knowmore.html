Quizzzz

-Let's see what you know..|
  • Quizzzz is a game that can shows you basic questions on mordern tech and as well on GK.
  • If you are an enthusiast who wants to have fun knowing n=basic concepts, then you found this right.
  • This game also keeps an track of your score and displays yur score accordingly.
  • User can choose any one of the options to proceed with the quiz.
  • Set of 3 questions are considered as 1 level and on clearing it, 3 points will be given.
  • There is a timer of 40 seconds set to all 3 questions.
  • Each question added one point to your score.
  • Each wrong answers would not deduct your score.
